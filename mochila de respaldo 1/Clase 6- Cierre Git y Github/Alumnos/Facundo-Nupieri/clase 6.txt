a. ¿Quién inventó el sistema de control de versión Git y por qué?
Linus Torvalds, creo Git, que es un software de control de versiones, pensando en la eficiencia, la confiabilidad y la compatibilidad del mantenimiento de versiones. Se creo porque linux tenía un gestor de paquetes que pasó a ser de pago, por lo que el desarrollador principal de Linux decidió crear un nuevo gestor de control de versiones para su sistema, y que diera soporte a su core.

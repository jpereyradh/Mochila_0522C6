A. Git fue desarrollado originalmente por Linus Torvalds, creador del kernel del sistema operativo Linux, en 2005. Torvalds lo desarrolló con la mentalidad de crear un nuevo gestor de control de versiones, bajo los pilares de eficiencia, confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente. Desde el 2009 y hasta la actualidad, el mantenimiento del software de Git está supervisado por Junio Hamano.

B. Actualmente, GitHub pertenece Microsoft. La empresa adquirió la plataforma en 2018, con el objetivo de adoptar GitHub como medio principal de control de versiones de Windows, acercándose además a la comunidad de desarrolladores y a la filosofía del código abierto.

C. Es posible trabajar con GitHub en el repositorio remoto de la página web y hacer modificaciones desde allí mismo, aunque es una alternativa poco cómoda en comparación al uso de la terminal, que además está integrada a la perfección con VSC. Así, codear, commitear y pushear se vuelven un proceso simple, cómodo y rápido.

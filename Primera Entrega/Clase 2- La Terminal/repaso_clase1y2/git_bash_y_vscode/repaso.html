Cuando hablamos de Vs Code y Gitbash nos referimos a interfaces donde podemos arrojar lineas de comando e interactuar con una computadora. Por un lado Vs Code nos proporciona el entorno con el que interactuaremos con el procesador, el teclado y el monitor, llamándolo así un entorno de desarrollo integrado y Gitbash nos proporciona el enotorno por donde comunicarnos con el sist. operativo a través de comandos escritos sencillos que ejecutan tareas o funciones.

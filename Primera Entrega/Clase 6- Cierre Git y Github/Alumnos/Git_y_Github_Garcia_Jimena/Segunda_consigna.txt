Crear carpeta - mkdir + (nombre carpeta)
Entrar en la carpeta creada - cd + (nombre carpeta)
Crear archivo - touch + (nombre archivo).formato (txt,pdf,doc,xls...)
Crear un nuevo repositorio - git init
Verificar el estado de los archivos/directorios - git status
Añadir un archivo especifico - git add + (nombre archivo)
Añadir todos los archivos - git add .
Commitear un archivo/directorio - git commit + (nombre archivo)-m "mensaje del commit"

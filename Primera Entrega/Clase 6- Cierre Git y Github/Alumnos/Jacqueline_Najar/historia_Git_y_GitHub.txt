a. ¿Quién inventó el sistema de control de versión Git y por qué?
Git fue creado en 2005 por Linus Torvalds, el creador del sistema opertativo 
Linux. Linux tenia un gestor de paquetes que paso a ser pago entonces surgio
la necesidad de crear su propio control de versiones para su sistema.

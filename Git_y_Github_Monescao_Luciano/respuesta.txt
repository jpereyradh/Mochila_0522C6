a. ¿Quién inventó el sistema de control de versión Git y por qué?
-Git es un software de control de versiones diseñado por Linus Torvalds, en 2005  la relación entre la comunidad que desarrollaba el kernel de Linux y la compañía que desarrollaba BitKeeper se vino abajo y la herramienta dejó de ser ofrecida de manera gratuita.
 Esto impulsó a la comunidad de desarrollo de Linux,  a desarrollar su propia herramienta teniendo como objetivo, la velocidad, un diseño sencillo, Gran soporte para desarrollo no lineal, completamente distribuido y Capaz de manejar grandes proyectos.

b. ¿A quién pertenece actualmente Github y por qué?

El 4 de junio de 2018, Microsoft compró GitHub por la cantidad de 7500 millones de dolares,la compañía ha adoptado una estrategia nueva, mucho más abierta al mundo del software libre y a la integración con otras plataformas y servicios.
ademas en plena explosión de cloud services (servicios en la nube), Microsoft tiene que competir con el líder en el mercado que era AWS, para tomar la cabeza la compania compro github para tener en  contacto directo con los desarrolladores que trabajan en esos servicios, ver sus necesidades y ofrecerles herramientas que faciliten su integración con Azure.
